JSON (JavaScript Object Notation) memiliki aturan penulisan yang sangat ketat. Berikut panduannya:

  • Gunakan Kutip Dua (") untuk semua Key (Nama Field) dan Value (Isi) yang berupa teks.
  • Dilarang menggunakan kutip satu (').
  • Dilarang ada Koma (,) pada baris terakhir sebelum tutup kurung }.
Format Template 
{
    "parameter": "value",
    "parameter2": "value2",
    "parentNestedParameter": {
        "childParameter": "childValue",
        "parentNestedParameter": {
            "childParameter": "childValue",
            "childParameter2": "childValue2"
        }
    }
}
